# proposal-bot

채용공고를 확인했을 때, 먼저 그 회사의 채용공고에 있는 아픈 지점을 찾고 미리 입력해놓은 내 경력, 도메인지식, 기술스택과 연관시켜서 해당 회사의 아픈 지점을 나의 어떤 능력으로 해결할 수 있는지 간략한 기술 제안 방향을 알려주는 크롬 확장프로그램.

추론, 예측하지 않고 명확하게 내 어떤 부분이 해당 회사의 아픈 지점을 해결할 수 있는지 그 근거를 제시해야 한다. 만약에 연관성이 낮다고 하더라도 어떻게든 나의 데이터를 바탕으로 연관지어서 기술제안을 해야하고 적합성이 높은지 그 확률도 계산해준다.

1. 미리 내 경력, 도메인지식, 기술스택을 입력받는다. 어떤 파일에 저장해두면 되겠다.
   나중에는 링크드인 API를 연동해서 사용자의 경력사항을 가져오거나
   건강보험 API를 사용해서 건강보험 자격득실 확인서를 통해 경력사항을 가져오거나 할 수 있다.
   현재는 어떤 파일이나 입력창에 사용자에게 데이터를 입력하라고 하면 된다.
   만약 주니어라 경력이 없다고 하면, 다른 직장이나 경험에 관련된 도메인 지식 또는 학교/학원 커리큘럼이라도 넣어야 한다.
2. 회사의 아픈 지점을 찾아내는 지침서 문서를 파일로 하나 만들어놓는다.
   이건 내가 따로 입력할 예정이니 파일로 만들기만 해라.
3. LLM API를 통해 1번과 2번을 통해서 해당 회사의 아픈 지점을 나의 어떤 능력으로 해결할 수 있는지 간략한 기술 제안 방향을 제시한다. 적합성이 얼마나 맞는지 퍼센트도 제시한다.
